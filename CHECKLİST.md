## Hazırlıq və ümumi tələblər
- [ ] Layihə məqsədi və hədəf istifadəçi təsdiqlənib  
- [ ] MVP funksiyaları və prioritetlər sənədləşdirilib  
- [ ] Staging mühiti HTTPS ilə qurulub və test alətləri mövcuddur

---

## Məzmun və hüquqlar
- [ ] Müəllif/naşir ilə ilkin lisenziya müqavilələri başlanılıb  
- [ ] İctimai domen kitabların siyahısı hazırlanıb və meta işarələnib  
- [ ] Hər kitab üçün **lisenziya statusu** və meta məlumat sahələri mövcuddur  
- [ ] Məzmun yükləmə prosesində lisenziya yoxlaması aktivdir

---

## Məhsul və oxucu funksiyaları
- [ ] Veb oxucu implementasiya edilib (səhifə çevirilməsi)  
- [ ] Font ölçüsü dəyişimi və gecə rejimi işləyir  
- [ ] Qeyd və vurğulama funksiyası istifadəçi hesabında saxlanır və sinxronizasiya olunur  
- [ ] Axtarış və kateqoriya filtr sistemi (müəllif, janr, populyar) aktivdir  
- [ ] Offline yükləmə funksiyası abunəçilər üçün məhdudiyyətlərlə işləyir

---

## Monetizasiya və ödənişlər
- [ ] Abunə paketləri və qiymət strukturu müəyyən edilib  
- [ ] Pay‑per‑book (tək kitab) checkout axını hazırlanıb  
- [ ] Ödəniş gateway sandbox inteqrasiyası test edilib (uğurlu və uğursuz kartlar)  
- [ ] Abunə aktivləşməsi və premium hüquqlar avtomatik tətbiq olunur

---

## Təhlükəsizlik və əməliyyat
- [ ] Bütün trafik HTTPS üzərindən yönləndirilir  
- [ ] İstifadəçi həssas məlumatları şifrələnmiş şəkildə saxlanır  
- [ ] Gündəlik backup planı və log yoxlaması aktivdir  
- [ ] DDoS və əsas təhlükəsizlik tədbirləri nəzərdən keçirilib

---

## Analytics və metriklər
- [ ] Event list tərtib edilib: session_start, page_turn, highlight, download_offline, purchase  
- [ ] Analytics inteqrasiyası aktivdir (DAU, MAU, DAU/MAU, churn, ARPU)  
- [ ] Dashboard-da 1 günlük və 30 günlük pəncərələr test edilib

---

## Müştəri dəstəyi və SLA
- [ ] Support e‑mail (support@lovable.dev) konfiqurasiya edilib və auto‑reply aktivdir  
- [ ] In‑app chat widget inteqrasiya edilib və ticket axını işləyir  
- [ ] SLA müəyyən edilib: ilk cavab < 24 saat; kritik məsələlər < 4 saat  
- [ ] Eskalasiya qaydası və hüquq/müəllif hüquqları üçün prioritet proses mövcuddur

---

## Test və buraxılış hazırlığı
- [ ] Yüksək prioritet user story-lar üçün bütün acceptance criteria test edilib  
- [ ] Ödəniş, analytics və dəstək inteqrasiyaları inteqrasiya testlərindən keçib  
- [ ] Bütün High severity defektlər həll edilib və ya mitigasiya sənədləşdirilib  
- [ ] MVP buraxılışı üçün son yoxlama və məhsul meneceri təsdiqi alınıb
